# 运行一个pull下来的vue-cli3项目步骤  

**第一步：`cnpm install`** 

**第二步：`cnpm install -g @vue/cli`**  

**第三步：`cnpm install -g @vue/cli-service-global`**  

**第四步：进入App.vue所在的目录后，运行`vue serve`**  


**发现pull下来的一个项目的目录框架跟我自己新建的不一样，就询问同伴。得知是vue-cli版本的区别。**
